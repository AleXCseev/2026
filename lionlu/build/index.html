Так, кросівки Lion-Lu ідеально створені для активного ритму життя. Завдяки м’якій підошві, яка ефективно амортизує кожен крок, навантаження на стопи значно зменшується. Це робить їх чудовим вибором як для довгих прогулянок містом, так і для вирішення щоденних справ. Універсальний чорний колір та поєднання класики з сучасним характером дозволяють носити їх практично з будь-яким одягом.
